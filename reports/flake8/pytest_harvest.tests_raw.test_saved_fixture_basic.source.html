26 """
27 Utility method to return the prefix to use for node ids, that will work whatever the invocation method.
28  
29 If this is not used the test might work when executed in "raw" mode, but might fail when executed using the
30 "pytester" plugin (from tests/test_all_raw_with_meta_check). Indeed the "pytester" plugin replaces the module
31 name by 'test_all_raw_with_meta_check.py'.
32  
33 IMPORTANT: the pytester plugin hangs instead of failing on windows, so a bug is hard to debug. The linux
34 version does not so a run on the travis CI usually helps.
35  
36 :param request:
37 :param test_func:
38 :return:
39 """
40 items = filter_session_items(request.session, filter=test_func.__module__)
41 last_node_id = items[-1].nodeid
42 prefix = last_node_id.split('::')[0]
43 return prefix
